This is a project named as Power EXhaust (PEX) Modelling Workshop which is an part of the course 01PUCND Nuclear Fusion Reactor Physics(NFRP). 
The NFRP is organized by Prof. Fabio Subba. Dr. Haosheng Wu is respondsable for PEX workshop.
This workshop include introduction about edege plasma physics, PEX modelling, SOLPS-ITER, and simple numerical analysis.
